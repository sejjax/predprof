# Предпроф экзамен 
Это реализация задач предпроф эказамена.
Здесь реализованы следующие задачи: 1 2 3 4 5 6

## Установка
```commandline
git clone https://github.com/sejjax/predprof
```
## Запуск заданий
```commandline
cd predprof
python ./1.py
python ./2.py
python ./3.py
python ./4.py
python ./5.py
```
или
```commandline
cd predprof
python3 ./1.py
python3 ./2.py
python3 ./3.py
python3 ./4.py
python3 ./5.py
```
